# BLE Indoor Tracking System (Concept Project)

This project is an offline BLE-based indoor tracking and alert system designed to monitor objects and movement across indoor environments.

## Overview

The system uses BLE beacon signals and gateway data to estimate room-level location, track movement history, and detect proximity events.

It was developed as a practical solution for real-world scenarios such as tracking mobile equipment or improving safety in indoor spaces.

## Features

- BLE-based tracking
- Room-level localization using RSSI signals
- Movement history tracking
- Last-seen and dwell-time logic
- Proximity-based alert system
- Offline-first architecture

## Technologies (used)

- C# / .NET
- ASP.NET Core
- SQLite
- BLE Beacons
- Teltonika Gateways
- REST APIs

## Note

Due to confidentiality and project ownership, the source code is not publicly available.

This repository is intended to present the concept, architecture, and technical approach of the system.
